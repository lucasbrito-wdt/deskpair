Features

Everything you need, nothing you don't.

A clean, modern control panel for your virtual display setup.

🖥

Real Second Monitor

Creates a virtual display that appears in GNOME Settings. Drag windows to it just like a physical screen.

🤚

Multi-Touch

Up to 10 simultaneous touch points. Pinch, scroll, and tap natively on touch-enabled VNC clients.

⚡

Low Latency

PipeWire-based frame capture with zero-copy buffer pools. Feels responsive, not like a slideshow.

📱

QR Code Connect

Scan the QR code from your tablet or phone to connect instantly. No typing IP addresses.

🔒

TLS & Auth

Generate TLS certificates and set up password authentication directly from the app. No terminal needed.

⚙

System Tray

Runs quietly in the background. Start on login, minimize to tray, control from the system indicator.

How It Works

Up and running in 3 steps.

The setup wizard handles everything automatically.

1

Install Deskpair

Download the .deb, .AppImage, or Flatpak. One click, no dependencies to hunt down.

2

Configure

Choose your resolution, port, and mode. The wizard checks your system and sets everything up.

3

Connect

Scan the QR code or type the address into any VNC client. Your new monitor is ready.

Compatibility

Works with your setup.

Wayland and X11, AMD and Intel. NVIDIA on Wayland too.

Wayland (Recommended)

Seamless virtual monitor via Mutter RemoteDesktop. No dialogs, no manual setup.

X11

Uses xrandr virtual outputs. Falls back to mirror mode on NVIDIA proprietary drivers.

VNC Clients

bVNC and AVNC (Android), RealVNC Viewer (iOS), Remmina and TigerVNC (Linux/Desktop).
